# DumbzoneClips

A tool that uses AI to analyze [The Dumbzone Podcast](https://www.thedumbzone.com) and pull clips of specific phrases or topics.

## Structure

Clips are organized by month, with each entry containing timestamped segments from the episode.

- `September/`, `October/`, `November/` — Monthly clip folders
- `index.html` — Web-based searchable index of all clips

## Usage

Browse clips via the `index.html` interface, which provides card and list views with search/filter by month or title.
